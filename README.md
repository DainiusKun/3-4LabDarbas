# 3-4LabDarbas
==============================================================\n
Programa kirta apskaičiuoti studentų vidurkį arba mediana.\n

Antra studentų namų darbų kiekis.\n

Ketvirta namų darbų rezultatai.\n
Penkta - egzaminų.\n
Galiausiai programa paklausia ar norite pamatyti mediana, ar vidurkį.\n
